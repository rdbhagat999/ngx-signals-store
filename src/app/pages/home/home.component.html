It is designed to provide a streamlined workflow for users and administrators, offering fine-grained access control and state management. The architecture leverages NGRX Signal Store for efficient state management, various route guards for security, and custom pipes and validators to enhance data handling.

Feature List

  • JWT-based authentication
  • NGRX Signal Store
  • CanActivate Auth Guard
  • CanActivate Admin Role Guard (Role-based Access Control)
  • CanDeactivate Guard
  • Custom Pipes
  • Custom Validators
